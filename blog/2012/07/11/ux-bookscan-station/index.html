[Ux] BookScan Station

Jul 11th, 2012

If you are going to impose a size limit on the scans that people want to e-mail to themselves…

“Email maximum size is 50MB.”

…then there should be a way to know how many bytes you’ve already used up with your present scans.

essential information, where you at?

Especially if the only warning you get when you go over is when you are just about to e-mail something. At this point, the only option left for users is to go back and delete scans that they’ve already done, which means they’ll need to be scanned again.
