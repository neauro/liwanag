Bunnies and Lozenges

Apr 18th, 2011


Exercise game, previously called “Bunnies and Monsters,” is moving smoothly.

Thanks for the lozenge, Teddy!

I believe to make the models Teddy used either Blender or Maya to create a .fbx file. At first I tried to copy the downloaded .fbx model into one of the assets folders, and then tried to make a prefab, but for some reason there was a problem generating colliders; things fared better when I
  1. Went to Assets > Import Package… and found the .fbx file
  2. Selected the imported model, then selected Generate Colliders in the FBXImporter tab of the Inspector
  3. Created a new prefab, then dragged the model into it
